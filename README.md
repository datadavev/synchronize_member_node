# synchronize_member_node

[![Actions Status][actions-badge]][actions-link]
[![PyPI version][pypi-version]][pypi-link]
[![PyPI platforms][pypi-platforms]][pypi-link]

Python library and CLI to synchronize a local copy by retrieving from the remote node using the DataONE API.

## Installation

```bash
python -m pip install synchronize_member_node
```

From source:
```bash
git clone https://github.com/datadavev/synchronize_member_node
cd synchronize_member_node
python -m pip install .
```
